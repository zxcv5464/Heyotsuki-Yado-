- [ ] `food_pool` 會平均分配給公池成員。

- [ ] `direct_staff` 有指定員工時分配給該員工。
- [ ] `direct_staff` 沒有指定員工時列入待分配清單。
- [ ] `dance_split` 未建立場次時列入待分配清單。
- [ ] `dance_split` 場次沒有參與者時列入待分配清單。
- [ ] `dance_split` 場次有參與者時平均分配。
- [ ] 數量為 1 且尚未建立場次的舞蹈品項，可一鍵批次建立空場次。
- [ ] 批次建立場次不會自動套用任何參與者。
- [ ] 舞蹈參與者使用 checkbox 操作，不需要 Shift/Ctrl 多選。
- [ ] 數量不等於 1 的舞蹈品項不會被自動批次建立，需手動新增場次。

- [ ] `excluded` 不產生薪資明細。
- [ ] `cancelled` 訂單不列入薪資。
- [ ] `deleted_at is not null` 訂單不列入薪資。
